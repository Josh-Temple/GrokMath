# Unit 14: Linear Equations and Inequalities with Rational Numbers

## Concept Map

- **Bridge from Unit 13:** Mixed rational operations now become tools for isolating unknown values in equations.
- **Equation structure:** A linear equation expresses balance between two equivalent quantities.
- **Inequality structure:** A linear inequality expresses a range of values rather than one exact value.
- **Operation logic:** Addition/subtraction and multiplication/division undo each other in controlled steps.
- **Connection forward:** These ideas prepare learners for systems, function models, and algebraic proof habits.

## Intuitive Explanation

In Unit 13, you practiced deciding and sequencing operations in multi-step rational contexts. Unit 14 keeps those same operations, but adds a new goal: find the unknown value that makes a statement true.

Think of an equation like a perfectly balanced scale. If you add, subtract, multiply, or divide one side, you must do the same action to the other side to keep the balance true. Solving is not guesswork; it is a sequence of valid balance-preserving moves.

Inequalities are similar, but they describe all values that satisfy a condition. For example, "at least" and "more than" naturally lead to intervals on the number line. A key detail is that multiplying or dividing an inequality by a negative number reverses the inequality direction.

## Formal Definition

A **linear equation in one variable** has the form

$$
ax + b = c,
$$

where $a, b, c \in \mathbb{Q}$ and $a \ne 0$. A value of $x$ is a solution if substituting it makes both sides equal.

A **linear inequality in one variable** has the form

$$
ax + b < c,\quad ax + b \le c,\quad ax + b > c,\quad \text{or} \quad ax + b \ge c.
$$

A value of $x$ is a solution if substitution makes the inequality true.

Valid transformation rules over $\mathbb{Q}$:

1. Add or subtract the same rational number on both sides.
2. Multiply or divide both sides by the same nonzero rational number.
3. For inequalities only: if multiplying or dividing by a negative number, reverse the inequality symbol.

## Worked Examples

### Example 1 - Foundational Equation

Solve $x - \frac{7}{4} = \frac{5}{2}$.

1. Add $\frac{7}{4}$ to both sides:
   $$
   x = \frac{5}{2} + \frac{7}{4}.
   $$
2. Use a common denominator:
   $$
   \frac{5}{2} = \frac{10}{4},\quad x = \frac{10}{4} + \frac{7}{4} = \frac{17}{4}.
   $$
3. Check: $\frac{17}{4} - \frac{7}{4} = \frac{10}{4} = \frac{5}{2}$, so the solution is valid.

### Example 2 - Standard Two-Step Equation

Solve $\frac{3}{5}x + 2.1 = -0.3$.

1. Subtract $2.1$ from both sides:
   $$
   \frac{3}{5}x = -2.4.
   $$
2. Divide by $\frac{3}{5}$ (equivalently multiply by $\frac{5}{3}$):
   $$
   x = -2.4 \div \frac{3}{5} = -2.4 \cdot \frac{5}{3} = -4.
   $$
3. Check: $\frac{3}{5}(-4) + 2.1 = -2.4 + 2.1 = -0.3$.

### Example 3 - Inequality with Negative Coefficient

Solve $-2x + 3 < 11$.

1. Subtract $3$ from both sides:
   $$
   -2x < 8.
   $$
2. Divide by $-2$ and reverse the inequality sign:
   $$
   x > -4.
   $$
3. Interpret: all values greater than $-4$ satisfy the inequality.

### Example 4 - Rational Inequality in Context

A savings balance after a fee is modeled by $s = 1.2x - 4.5$, where $x$ is weekly deposit amount in dollars. Find all $x$ such that $s \ge 9$.

1. Write and solve:
   $$
   1.2x - 4.5 \ge 9 \Rightarrow 1.2x \ge 13.5.
   $$
2. Divide by positive $1.2$ (direction stays the same):
   $$
   x \ge \frac{13.5}{1.2} = 11.25.
   $$
3. Interpretation: deposits must be at least $11.25$.

## Common Mistakes and How to Avoid Them

- **Mistake:** Applying an operation to only one side of an equation.
  **How to avoid it:** Treat equations as balance statements and always perform each transformation on both sides.
- **Mistake:** Forgetting to reverse an inequality after multiplying or dividing by a negative value.
  **How to avoid it:** Mark the negative operation step explicitly and immediately flip the symbol.
- **Mistake:** Arithmetic slips with fractions/decimals while solving.
  **How to avoid it:** Convert to a consistent form (fraction or decimal) before combining terms and check each intermediate step.
- **Mistake:** Stopping after finding a value without checking.
  **How to avoid it:** Substitute the candidate solution into the original equation/inequality to verify correctness.

## Quick Checks

1. Solve: $x + \frac{3}{2} = \frac{11}{6}$.
   - **Answer:** $x = \frac{11}{6} - \frac{9}{6} = \frac{2}{6} = \frac{1}{3}$.
2. Solve: $0.4x - 1.8 = 0.6$.
   - **Answer:** $0.4x = 2.4$, so $x = 6$.
3. Solve: $-\frac{5}{2}x \ge 10$.
   - **Answer:** Divide by $-\frac{5}{2}$ and reverse sign: $x \le -4$.
4. True or false: if $3x - 2 < 7$, then $x < 3$.
   - **Answer:** True, because $3x < 9$ and dividing by positive $3$ keeps direction.

These checks reinforce the core routine: isolate the variable using valid inverse operations, track sign rules carefully, and verify results in the original statement.

## Graded Exercises

### Level 1 - Warm-up

1. Solve: $x - \frac{5}{3} = \frac{1}{6}$.
2. Solve: $\frac{7}{4}x = -\frac{21}{8}$.
3. Solve: $2.5x + 1 = 6$.

### Level 2 - Core Practice

4. Solve: $\frac{2}{3}x - \frac{1}{2} = \frac{5}{6}$.
5. Solve and check: $-1.2x + 3.4 = -2.6$.
6. Solve: $4 - 3x \le 10$.
7. A taxi fare model is $F = 2.75m + 3.50$, where $m$ is miles. For fares at most $17.25$, find all possible $m$.

### Level 3 - Challenge

8. Solve: $\frac{3}{4}(2x - 5) = \frac{1}{2}(x + 7)$.
9. Solve: $-\frac{2}{5}(x - 3) > \frac{4}{3}$.
10. A tank has $18$ liters and drains at $0.6$ liters per minute. For at least $9$ liters remaining, what times $t$ satisfy the condition?

3. $x = 2$.
4. $x = 2$.
5. $x = 5$.
6. $x \ge -2$.

## Summary and Key Takeaways

Unit 14 translates rational-operation fluency into algebraic solving habits. The central idea is structural: use inverse operations to isolate the variable while preserving a true statement on both sides.

- Linear equations give exact solution values; linear inequalities give sets of values.
- Equation-solving steps must preserve balance by acting on both sides equally.
- In inequalities, multiplying or dividing by a negative reverses the inequality symbol.
- Fraction/decimal fluency from Units 12-13 directly supports accurate solving in Unit 14.
- These methods prepare learners for systems of equations, linear modeling, and function-based reasoning.
